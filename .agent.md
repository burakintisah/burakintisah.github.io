## ğŸ› Troubleshooting Guide

### Build Issues

**Problem**: TypeScript compilation errors
```bash
# Solution
npm run lint              # Check for errors
npx tsc --noEmit          # Type check only
```

**Problem**: Build fails with dependency errors
```bash
# Solution
rm -rf node_modules package-lock.json
npm install
npm run build
```

### Routing Issues

**Problem**: 404 error on page refresh (GitHub Pages)
- **Cause**: SPA routing not configured
- **Solution**: Verify `public/404.html` exists and contains redirect script
- **Check**: CNAME file exists in `public/` folder

**Problem**: Routes not working locally
```bash
# Solution: Use hash router or configure Vite
# Current: Using BrowserRouter with 404.html workaround
```

### Firebase Issues

**Problem**: Analytics not tracking
```bash
# Check list
1. Verify .env file exists and has correct Firebase config
2. Check browser console for errors
3. Verify Firestore security rules allow writes
4. Test Firebase connection in browser DevTools
```

**Problem**: "Permission denied" in Firestore
```javascript
// Update Firebase Console â†’ Firestore â†’ Rules
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /analytics/{document} {
      allow read, write: if true; // For development
    }
  }
}
```

### Styling Issues

**Problem**: Dark mode not working
```bash
# Check list
1. Verify dark mode toggle in Layout.tsx
2. Check localStorage persistence
3. Ensure all components have dark: variants
4. Test with browser DevTools
```

**Problem**: Tailwind classes not applied
```bash
# Solution
npm run dev  # Restart dev server
# Or check tailwind.config.js content paths
```

### Deployment Issues

**Problem**: `npm run deploy` fails
```bash
# Check list
1. Verify gh-pages package installed
2. Check git remote configured
3. Ensure clean working directory
4. Try manual: git push origin gh-pages
```

**Problem**: Site not updating after deploy
```bash
# Solution
1. Clear browser cache (Ctrl+Shift+R)
2. Wait 5-10 minutes for GitHub Pages
3. Check GitHub repo â†’ Settings â†’ Pages
4. Verify gh-pages branch exists
```

### Performance Issues

**Problem**: Slow page load
```bash
# Debug
1. Run Lighthouse audit
2. Check Network tab for large files
3. Optimize images (see Photo Management)
4. Review bundle size in build output
```

**Problem**: Large bundle size
